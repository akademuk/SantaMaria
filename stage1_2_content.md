# Концепція та Структура (Етап 1)

**Концепція:** "Santa Maria: Гармонія Природи та Здоров'я".
Дизайн фокусується на цілющій силі природи. Кольорова палітра натхненна Карпатами: глибокий лісовий зелений, блакитний колір мінеральних вод та теплі земляні відтінки.
Верстка простора, з використанням "повітря" для створення відчуття спокою (premium feel).
Типографіка: вишуканий засічковий шрифт для заголовків та чистий гротеск для основного тексту.
